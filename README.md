# ProyectoIngenieria22

 Este repositorio contiene el proyecto de ingeniería del analisis financiero de los datos de yfinance de apple hecho a medida para los inversores, este se divide en dos partes principales: un análisis financiero de la acción de Apple y una segmentación de datos aplicada a un conjunto de clientes.
En la primera parte del proyecto se realiza un análisis financiero de Apple Inc. (AAPL) utilizando datos reales obtenidos a través de la librería yfinance. Se estudia la evolución histórica del precio de la acción desde 2018 hasta 2025, analizando tanto el precio de cierre como el volumen de negociación. Para que sea más atractivo visualmente, se utilizan diferentes gráficos y escalas, incluyendo medias móviles que permiten observar mejor las tendencias del precio a lo largo del tiempo.
Además del análisis individual de Apple, se incluye una comparación con otros activos del mercado como Microsoft (MSFT) y el índice S&P 500 (SPY). Esta comparación sirve para situar el comportamiento de Apple dentro del contexto del mercado y comprobar cómo se comporta frente a otras referencias importantes. Para ello se utilizan precios normalizados, rendimientos acumulados y una matriz de correlación entre activos.
A partir de estos datos, se desarrolla un modelo sencillo de predicción del precio de Apple utilizando XGBoost. El modelo se basa en datos históricos del precio mediante ventanas temporales (lags) y se utiliza para estimar el precio de la acción a distintos horizontes temporales. En concreto, se realizan predicciones a 1, 5, 10 y 15 días, que se muestran mediante gráficos individuales y una gráfica conjunta con el histórico reciente. Estas predicciones sirven únicamente como aproximación y no deben entenderse como valores exactos del mercado.
La segunda parte del proyecto está dedicada a la segmentación de datos. Para ello se genera un dataset simulado de clientes, con variables habituales en estudios de marketing como la frecuencia de compra, el gasto medio, los días desde la última compra, la satisfacción del cliente y el nivel de fidelización. El objetivo de esta parte es aprender a identificar distintos perfiles de comportamiento a partir de los datos.
Como resultado, se identifican diferentes perfiles de clientes, como clientes premium, clientes dormidos y clientes ocasionales.
Los resultados de la segmentación se representan mediante distintos gráficos, como gráficos PCA, gráficos radar y comparaciones de KPIs entre perfiles. Estas visualizaciones permiten entender mejor las características de cada grupo y relacionarlas con posibles decisiones de marketing o estrategia comercial.
En conjunto, este proyecto sirve como una primera aproximación práctica al análisis de datos aplicado a problemas reales y simulados.
